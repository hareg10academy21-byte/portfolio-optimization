# 📊 Portfolio Optimization & Forecasting Project

## 📌 Overview
This project focuses on financial time series forecasting and portfolio optimization using:

- ARIMA/SARIMA (statistical modeling)
- LSTM (deep learning forecasting)
- Risk metrics and portfolio analysis

Assets:
- TSLA (Tesla)
- SPY (S&P 500 ETF)
- BND (Bond ETF)

---

## 🚀 Project Structure

- Task 1: Data preprocessing & EDA
- Task 2: Forecasting models (ARIMA + LSTM)
- Task 3: Portfolio optimization (Efficient Frontier)
